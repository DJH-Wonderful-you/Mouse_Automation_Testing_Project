# 打包命令

## 问题说明

如果直接使用 `.\.venv\Scripts\pyinstaller.exe` 遇到 "Fatal error in launcher" 错误，这是因为虚拟环境中的可执行文件路径引用问题。

## 解决方案

使用 `python -m PyInstaller` 代替直接调用 `pyinstaller.exe`。

可执行程序（目录版）：

```powershell
.\.venv\Scripts\python.exe -m PyInstaller --noconfirm --clean --windowed --name Baseband_Test_Data_Statistics_Tool --icon="src/resources/app_icon.ico" --add-data "src/resources;src/resources" main.py
```

安装包版：

```powershell
& 'C:\Users\DJH\AppData\Local\Programs\Inno Setup 6\ISCC.exe' 'installer.iss'

& 'D:\Users\2507140123\AppData\Local\Programs\Inno Setup 6\ISCC.exe' 'installer.iss'
```

在 cmd 里可直接这样执行（项目根目录下）：

```cmd
cd /d D:\RJHZ\Project_Code\Python\Baseband_Test_Data_Statistics_Tool

.\.venv\Scripts\python.exe -m PyInstaller --noconfirm --clean --windowed --name Baseband_Test_Data_Statistics_Tool --add-data "src/resources;src/resources" main.py

"C:\Users\DJH\AppData\Local\Programs\Inno Setup 6\ISCC.exe" installer.iss
```
